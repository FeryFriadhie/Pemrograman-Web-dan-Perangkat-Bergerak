C. Landasan Teori

1.Anatomi hyperlink

Berdasarkan lokasi atau alamat dokumen yang diakses,link pada HTML dibedakan menjadi tiga jenis, yaitu link absolute, link relative, dan link ke bagian dokumen tertentu. Jadi, perbedaannya terletak pada nilai atribut HREF-nya saja.

a. Link absolute

Link yang akan menunjuk ke halaman dari situs web lain. Penulisan alamatnya harus lengkap, contoh: W3 Pada contoh diatas link akan menuju ke situs web http://www.w3.org/.

b. Link relative

Link yang tujuannya mengarah ke dokumen-dokumen lain yang masih berada di dalam satu situs web yang sama (komputer yang sama). Penulisan alamat tujuannya pun tidak perlu ditulis secara lengkap, cukup nama dokumennya saja dan nama direktorinya (jika ada).

Contoh:
Kontak
-> digunakan untuk memanggil file html dalam satu direktori yang sama

Lihat Foto Jeep Wrangler Sahara
-> digunakan untuk memanggil file html dalam direktori yang berbeda

Contoh: Format link berupa teks:

C. Link ke bagian dokumen tertentu (internal link)

Format link antar isi pada suatu halaman web biasa disebut juga dengan link dalam satu halaman (internal link). Misalnya membuat tautan link untuk mengarah ke atas, ke tengah, atau ke bawah. Link ini dibuat untuk dokumen panjang sekali, sehingga apabila ditampilkan dalam browser akan mengharuskan pengguna melakukan scroll layar berulang-ulang.

Navigasi untuk penelusuran dokumen dapat dimudahkan dengan membuat link atau bagian. Caranya adalah menandai setiap bagian tersebut dengan memberinya nama.

Cara membuat link dalam satu halaman:

1. Menentukan nama target Pilih bagian artikel yang ingin dijadikan sebagai target link, bisa berupa paragraph, subjudul, kalimat, atau kata yang diinginkan. Kemudian buat nama atau id pada kata tersebut dengan satu nama variabel. Pemberian nama tidak menggunakan spasi.

Contoh 1:
BAB III
Contoh 2:
BAB IV

2. Membuat link target Buatlah kalimat atau kata yang ingin dijadikan sebagai link yang akan ditujuan ke target link.

Contoh:
Ke Bab III
Ke Bab IV

3. Format target link, email, telepon, sms, dan download

a. Format target link
Jika link di klik maka browser akan menampilkan halaman yang dituju oleh link tersebut. Ada 2 macam tampilan oleh browser dalam memunculkan halaman tujuan tersebut.

1.) Pertama, ditampilkan pada jendela yang sama sehingga halaman tempat link itu ditutup dan digantikan oleh halaman yang dituju, dan ini merupakan cara pemunculan default.

2.) Kedua, ditampilkan pada jendela baru sehingga akan muncul jendela baru untuk menampilkan halaman yang dituju.

Dalam pembuatan link ini harus ditambahkan atribut TARGET = “_blank” dalam tag , sehingga penulisannya adalah sebagai berikut:
 Teks / Gambar
Keterangan:

- NamaDokumen : nama dokumen yang akan ditampilkan jika link di klik
-NamaTarget : target atau tempat tujuan dari dokumen yang akan ditampilkan

Value atau nilai dari target bisa bernilai _blank maupun nama frame (jika layar web anda menggunakan frame).
Berikut beberapa nilai target yang dapat diisikan pada link:

1.)_blank : halaman yang dipanggil akan muncul pada jendela baru
2.)Top : halaman yang dipanggil akan mengisi penuh jendela yang sama, meskipun tadinya ada frame.
3.)_self : halaman yang dipanggil akan mengisi tempat yang sama bila link tersebut dalam sebuah frame, maka halaman yang dipanggil akan mengisi frame tersebut.
4.)_parent : halaman yang dipanggil akan mengisi frame set satu tingkat lebih tinggi dari frame tempat link yang bersangkutan.

b. Link email

Link email dapat digunakan untuk mengirim ke alamat email. Jika link ini di klik, maka akan membuka jendela pengiriman email dari program email yang terinstal pada computer (misalnya Microsoft Outlook). Pada kotak tujuan email akan tercantum alamat email tujuan. Contohnya, link yang langsung menuju ke alamat email, dengan penulisan kode seperti berikut:

Kirim email

C. Link telepon

Link telepon dapat digunakan untuk melakukan panggilan telepon. Jika link ini di klik, maka akan melakukan panggilan telepon pada nomor tujuan. Contoh penulisan kodenya seperti berikut:

klik disini untuk menghubungi saya

d. Link SMS

Link SMS dapat digunakan untuk melakukan pengiriman pesan SMS pada nomor tujuan. Jika link di klik, maka akan melakukan pengiriman SMS pada nomor tujuan. Contoh penulisan kodenya seperti berikut:

SMS ke saya:

e.Link Download

Link download digunakan untuk pengambilan file dari server atau yang lebih di kenal dengan istilah download. Link ini bertujuan untuk file-file yang bukan format file web, yang tidak dapat ditampilkan oleh browser. Seperti aplikasi EXE, File compress ZIP, dokumen DOC, XLS, dll. Jika file tersebut di klik maka akan muncul kotak dialog yang menanyakan apakah anda ingin menyimpan file tersebut dalam hardisk atau membukanya dengan program yang terinstal pada computer yang digunakan. Contoh penulisan kodenya sebagai berikut:

download

klik DOWNLOAD untuk mengunduh lampiran
